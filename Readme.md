### TextBlog
This is a text blogging app.
- Users can register, write posts, and comment.
- In the future, the ability to restore access via email and the ability to share publications via email will be added.
- You can add tags to publications and recommend similar publications based on common tags. 
- You can view the most commented and most viewed posts.
- There is a site map to improve SEO functionality.
- The application uses a full-text search engine using PostgreSQL
- The demo version is located on the temporary render website.
